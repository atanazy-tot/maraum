# Tech Stack

## Frontend - Astro with React for interactive components:

- Astro 5 allows for creating fast, efficient websites and applications with minimal JavaScript

- React 19 provides interactivity where needed

- TypeScript 5 for static typing of code and better IDE support

- Tailwind 4 enables convenient styling of applications

- Shadcn/ui provides a library of accessible React components that we will base our UI on

## Backend - Supabase (cloud managed) as a comprehensive backend solution:

- Provides a PostgreSQL database

- Provides SDKs in multiple languages that will serve as Backend-as-a-Service

- Cloud managed version for the MVP stage (can be self-hosted later if needed)

- Has built-in user authentication

## AI - Communication with Claude through the Streaming API:

- Uses Claude Streaming API for direct access to Anthropic's language models

- Provides real-time streaming responses for improved user experience

## CI/CD and Hosting:

- Github Actions for creating CI/CD pipelines

- DigitalOcean for hosting the application via Docker image
